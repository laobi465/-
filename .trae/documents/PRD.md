## 1. 产品概述

「PIXEL MEMORY」是一款复古像素风格的记忆翻牌小游戏,玩家需要在限定时间或步数内翻开卡片找到相同图案的配对。游戏致敬 8-bit 街机时代,采用像素化图形、CRT 扫描线效果和 chiptune 音效,带来怀旧而富有挑战性的益智体验。

- 主要目的:提供一款轻量、上头、即开即玩的记忆挑战游戏,锻炼短期记忆力
- 目标用户:怀旧游戏爱好者、休闲益智游戏玩家、碎片时间消遣人群
- 产品价值:通过难度分级和最佳记录系统提升复玩率,用复古美学打造差异化记忆点

## 2. 核心功能

### 2.1 用户角色
无需区分用户角色,所有访客均为玩家,数据通过 localStorage 本地存储。

### 2.2 功能模块
1. **主游戏页**:标题区、游戏面板(卡片网格)、状态栏(步数/时间/配对数)、控制按钮
2. **难度选择**:开始前选择简单(4×4)、中等(6×6)、困难(6×8)三档难度
3. **结算页**:胜利弹窗,展示成绩、最佳记录、重新开始按钮

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 主游戏页 | 标题区 | 像素风 LOGO "PIXEL MEMORY",闪烁霓虹效果 |
| 主游戏页 | 难度选择栏 | 三档难度按钮(简单/中等/困难),显示对应网格规格和最佳记录 |
| 主游戏页 | 状态栏 | 实时显示步数、计时器、已配对数/总配对数 |
| 主游戏页 | 游戏面板 | 卡片网格,点击翻转,配对成功保持翻开,失败翻回 |
| 主游戏页 | 控制按钮 | 重新开始、暂停/继续、音效开关 |
| 主游戏页 | 结算弹窗 | 胜利时弹出,展示用时、步数、是否破纪录、再玩一次按钮 |

## 3. 核心流程

玩家进入游戏 → 选择难度 → 卡片随机洗牌背面朝上 → 玩家依次翻开两张卡片 → 若图案相同则配对成功保持翻开 → 若不同则短暂展示后翻回 → 重复直到所有卡片配对完成 → 弹出胜利结算 → 记录最佳成绩 → 可选择再玩一次或更换难度。


## 4. 用户界面设计

### 4.1 设计风格
- **主色调**:深紫黑背景 (#0d0221) + 霓虹粉 (#ff2a6d) + 青蓝 (#05d9e8) + 明黄 (#f9f871) 点缀
- **辅助色**:复古绿 (#00ff9f)、像素灰 (#2d1b3d)
- **按钮风格**:像素方块按钮,3D 凸起边框(上左亮色、下右暗色模拟立体),点击有按压动效
- **字体**:"Press Start 2P" 像素字体作为标题/数字,"VT323" 等宽像素字体作为正文
- **布局**:居中卡片网格,顶部标题,底部状态栏,整体 CRT 显示器边框包裹
- **图标/emoji 风格**:像素化 8-bit 图案作为卡片正面(如像素心、像素星、像素蘑菇等)

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|-----------|-------------|-------------|
| 主游戏页 | 标题区 | 霓虹发光像素 LOGO,扫描线动画,抖动入场效果 |
| 主游戏页 | 难度选择栏 | 三档像素按钮,hover 发光,显示星级和最佳记录 |
| 主游戏页 | 状态栏 | 像素边框数字显示器,步数/时间/配对实时跳动 |
| 主游戏页 | 游戏面板 | 卡片像素边框,翻转 3D 动画,配对成功闪光特效 |
| 主游戏页 | 控制按钮 | 像素图标按钮,重新开始/暂停/音效开关 |
| 主游戏页 | 结算弹窗 | 像素边框弹窗,VICTORY 闪烁文字,成绩列表,破纪录烟花特效 |
| 主游戏页 | CRT 滤镜 | 全屏扫描线 + 边角暗角 + 轻微色偏,模拟老式 CRT 显示器 |

### 4.3 响应式
- 桌面优先设计,游戏面板居中适配
- 移动端自适应:卡片网格等比缩放,触摸点击优化,状态栏改为顶部横向排列
- 困难模式在移动端自动缩小卡片尺寸保证可用性

### 4.4 动效设计
- 卡片翻转:CSS 3D transform 翻转动画 (rotateY)
- 配对成功:卡片发光脉冲 + 缩放弹跳
- 配对失败:卡片红色闪烁后翻回
- 胜利结算:弹窗从中心放大入场,文字逐字打字机效果
- 背景:星空粒子缓慢漂移,扫描线持续滚动
